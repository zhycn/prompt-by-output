# prompt-by-output

> **用真实输出打磨每一个提示词 —— 名字 + 内容，即刻保存，持续进化。**

这个仓库是我个人的提示词库。  
每个提示词都经过实际使用（或即将使用），以 **GitHub Issue** 形式保存：

- **标题** = 提示词名称（一句话说清用途）  
- **正文** = 提示词完整内容（可直接复制使用）

没有花哨格式，没有强制输出记录——先让好点子留下来，跑不跑、效果如何，以后再说。

## 🚀 怎么贡献？
1. 开个 Issue  
2. 标题写清楚这是什么提示词  
3. 正文粘贴提示词内容  
4. Done.

> 志向可以远大，起步必须简单。  
> 先存下来，就是胜利 ✅
